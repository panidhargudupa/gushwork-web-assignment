# Web Assignment – HDPE Pipes Product Page

This project is a **responsive product landing page** built as part of the **Gushwork web assignment**.

The page has been developed using **pure vanilla HTML, CSS, and JavaScript** without using any external frameworks or libraries, as per the assignment requirements.

## 📌 Features

* Responsive **desktop and mobile layout**
* Product image gallery / slider
* Sticky price bar
* Request quote modal
* Download technical datasheet modal
* Technical specifications section
* Clean and readable code structure
